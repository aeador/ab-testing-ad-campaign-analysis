# A/B Testing Analysis: Ad Campaign Effectiveness

## 📊 Overview
This project analyzes the effectiveness of a digital advertising campaign by comparing conversion rates between a control group and a test group.

## 🎯 Objective
To determine whether the ad campaign led to a statistically significant increase in conversions.

## 🧠 Methods
- Data cleaning and preprocessing
- Conversion rate calculation
- Hypothesis testing (t-test)
- Statistical significance evaluation

## 🛠️ Tools Used
- Python (pandas, scipy) / R
- Jupyter Notebook

## 📈 Key Results
- Control conversion rate: 30%  
- Test conversion rate: 70%  
- Lift: 133%  
- P-value: 0.081 (greater than 0.05, not statistically significant)

## 💡 Conclusion
The analysis provides insights into whether the ad campaign had a measurable impact on user behavior and conversion outcomes.
